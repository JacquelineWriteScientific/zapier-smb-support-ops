# Zapier SMB Customer Support Ops

This repository contains reference Zapier automations designed for small businesses, creators, and service providers.

The focus is on:
- Clean, understandable Zap builds
- Human-centered customer communication
- Build-and-handoff delivery (not managed services)

Each automation includes:
- Build steps (Zapier terminology)
- Required app connections
- Failure points
- Client handoff guidance

----
## Scope and Delivery Model

This project follows a build-and-handoff model.

### What is included
- Zap design and configuration
- Field mapping and testing
- Documentation for client ownership
- 7-day stabilization window

### Done does not mean
- Ongoing optimization
- New features
- Additional apps
- Strategy changes

Requests outside the original build are scoped as new work / new contracts. :)
